## ✨ Features

- 🎁 **Gift & Event Tracking**: Organize gifts, contacts, and important dates
- 🔔 **Smart Notifications**: Automated reminders for upcoming events
- 🖼️ **Image Uploads**: Secure S3 storage with pre-signed URLs
- 🔒 **Authentication**: Secure login with AWS Cognito
- 📱 **Mobile-Ready**: Responsive, modern UI

## 🏁 Quick Start

```bash
# 1. Clone the repo
https://github.com/Jeanfr1/marie-mobile-canvas.git
cd marie-mobile-canvas

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

---

## 🌍 Deploy to Netlify (Recommended)

1. [Sign up for Netlify](https://app.netlify.com/signup) and connect your GitHub repo.
2. Set build command: `npm run build`
3. Set publish directory: `dist`
4. Click **Deploy** and go live!
